# Overview

`ai-engineering-harness` is a minimal TypeScript scaffold for small engineering projects. It combines strict TypeScript, linting, tests, CI, security scanning, dependency updates, and documentation templates for AI-assisted development.

The repository is intentionally generic. It is not a framework, platform, product starter, or agent runtime. It gives a project enough structure to support careful implementation and repeatable verification while leaving application choices to the project owner.

## What It Provides

- Node.js 24 and npm project setup
- TypeScript with strict compiler settings
- ESLint flat config
- Vitest tests
- GitHub Actions verification
- CodeQL analysis
- Dependabot updates
- AI collaboration and engineering documentation templates

## Intended Use

Use this harness when starting a small project where requirements, assumptions, trade-offs, and validation should remain visible. Replace the example implementation with project-specific code and keep the verification workflow intact.
